Full_Body, created by vi from V-OS.
Full_Body is a web application made to help organize and sequence fitness exercises created to make time spent exercising more engaging.

This application uses cookies to save your workout plan, meaning that in the event that your browser or computer changes, or deletes these cookies, your workout will unfortunately be erased. This is my way of avoiding using a database while still giving different users the ability to have a unique experience.

I hope you enjoy using this as much as I do! If you'd like to know more about this app, please check out the github page. And if you'd like to know more about my other work, you can also take a look at my website, v-os.
Full_Body is an exercise sequencer that helps keep your workouts varied, and makes exercising easier and more motivating through time tracking and a pleasant interface.
